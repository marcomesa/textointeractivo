INTRODUCCIÓN

En este simulador se podrá interactuar así: si se abre la llave el tanque se comenzará a llenar; si se inicia el vaciado es como si se abriera una válvula en la parte inferior del tanque. Se puede cerrar la llave, parar el vaciado o reiniciar el simulador para empezar de nuevo. A la derecha aparecerá una gráfica que indicará el comportamiento del proceso. Debajo se encontrará el comportamiento de las variables de volumen y tiempo y siguiendo más abajo, se encontrará unos controles para ampliar o reducir el tanque, también para aumentar la velocidad de llenado o vaciado del mismo.

OBJETIVOS

Comprender, a través de un simulador de llenado y vaciado de tanques, la derivada como razón de cambio.

INSTRUCCIONES

Utiliza los botones para abrir o cerrar las llaves de llenado o vaciado.

Las tres barras permiten variar las velocidades de llenado o vaciado o cambiar el ancho del tanque.
